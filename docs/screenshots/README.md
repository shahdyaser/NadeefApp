## Screenshots (fresh, from the app)

Put **new screenshots captured from the running app** in this folder.

The user guide expects these filenames:

- `01-signin.png`
- `02-signup.png` (optional if same screen as sign-in)
- `03-setup-house.png`
- `04-home.png`
- `05-profile-invite.png`
- `06-add-room.png`
- `07-room-details.png`
- `08-add-task.png`
- `09-all-tasks.png`
- `10-task-details.png`
- `11-skip-popup.png`
- `12-completed-undo.png`

After you add them, open `USER_GUIDE.md` to verify images render.
